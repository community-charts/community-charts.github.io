# n8n chart version 1.24.14 released!

We are excited to share the release of the **n8n Helm chart version 1.24.14** as of July 18, 2026, which includes **app version 2.30.7**. This update is packed with new features and community-driven enhancements, all designed to streamline your deployment of n8n on Kubernetes.

## What's New in n8n 1.24.14 chart

This release not only updates the Helm chart (version 1.24.14) but also the underlying application (version 2.30.7). Here's a summary of the key changes:

- **Changed**: Updated the n8nio/n8n image version to 2.30.7
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a complete list of changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.14) on GitHub.
