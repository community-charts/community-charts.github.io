# Outline Chart Version 0.7.5 Released!

We are excited to share the launch of the **Outline Helm chart version 0.7.5** on June 2, 2026, now featuring **app version 1.8.0**. This release brings a host of new features, enhancements, and improvements driven by our vibrant community, all designed to make deploying Outline on Kubernetes easier than ever.

## What's New in Outline 0.7.5 Chart

In this update, you’ll find significant improvements to both the Helm chart (version 0.7.5) and the underlying application (version 1.8.0). Here’s a quick overview of the changes:

- **Changed**: Updated the Outline image version to 1.8.0
    - [Upstream Project](https://hub.docker.com/r/outlinewiki/outline)
- **Changed**: Updated the PostgreSQL dependency from 18.6.8 to 18.7.0
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a detailed look at all changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/outline-0.7.5) on GitHub.
