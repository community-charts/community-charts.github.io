# mlflow chart version 1.8.4 launched!

We are excited to announce the launch of the **mlflow Helm chart version 1.8.4** on June 2, 2026, which includes **app version 3.13.0**. This update introduces a range of new features, enhancements, and valuable community-driven improvements to make deploying mlflow on Kubernetes even more seamless.

## What’s New in mlflow 1.8.4 chart

This release highlights updates to the Helm chart (version 1.8.4) alongside the underlying application (version 3.13.0). Here’s a brief overview of the changes:

- **Changed**: Updated the burakince/mlflow image version to 3.13.0
    - [Explore the Upstream Project](https://hub.docker.com/r/burakince/mlflow)
- **Changed**: Updated the postgresql dependency from 18.6.10 to 18.7.0
    - [View on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a comprehensive list of changes, visit the [release notes](https://github.com/community-charts/helm-charts/releases/tag/mlflow-1.8.4) on GitHub.
