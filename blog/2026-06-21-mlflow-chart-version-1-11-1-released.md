# mlflow chart version 1.11.1 released!

We are excited to share the launch of the **mlflow Helm chart version 1.11.1** on June 21, 2026, featuring **app version 3.14.0**. This update introduces an array of new features, enhancements, and community-driven improvements aimed at simplifying your experience deploying mlflow on Kubernetes.

## What's New in mlflow 1.11.1 chart

This release encompasses updates to the Helm chart (version 1.11.1) as well as the underlying application (version 3.14.0). Here’s a brief overview of the changes:

- **Changed**: Updated oauth2-proxy image version to v7.15.3
    - [oauth2-proxy](https://quay.io/repository/oauth2-proxy/oauth2-proxy)
- **Changed**: Updated dependency postgresql from 18.7.5 to 18.7.6
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a comprehensive list of updates, check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/mlflow-1.11.1) on GitHub.
