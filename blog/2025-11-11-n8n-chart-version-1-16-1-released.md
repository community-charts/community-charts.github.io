# n8n chart version 1.16.1 released

We’re excited to announce the release of the **n8n Helm chart version 1.16.1**, now available as of November 11, 2025. This update includes **app version 1.119.1**, bringing fresh enhancements and performance improvements to help you easily deploy and manage n8n on Kubernetes.

## What’s New in Helm Chart 1.16.1

This release updates both the Helm chart and the underlying app version, incorporating key community contributions and dependency upgrades:

- 🔄 **Updated**: Base image upgraded to `n8nio/n8n:1.119.1`  
  → [View Project on GitHub](https://github.com/n8n-io/n8n)
- 🔧 **Updated**: Redis chart bumped from `23.2.6` to `23.2.12`  
  → [Redis via ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)
- 🛠️ **Updated**: PostgreSQL chart upgraded from `18.1.4` to `18.1.8`  
  → [PostgreSQL via ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For the full list of changes, check out the [n8n 1.16.1 release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.1) on GitHub.
