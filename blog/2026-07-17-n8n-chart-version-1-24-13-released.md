# n8n chart version 1.24.13 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.13** on July 17, 2026, featuring the all-new **app version 2.30.6**. This update introduces an array of new features, enhancements, and valuable community-driven improvements, all designed to streamline the deployment of n8n on Kubernetes.

## What's New in n8n 1.24.13 chart

This release encompasses updates to both the Helm chart (version 1.24.13) and the accompanying application (version 2.30.6). Here’s what you can expect:

- **Changed**: Updated the n8nio/n8n image version to 2.30.6
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a comprehensive list of changes, feel free to check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.13) on GitHub.
