# mlflow chart version 1.8.6 released!

We are excited to share the release of the **mlflow Helm chart version 1.8.6** on 2026-06-18, featuring **app version 3.14.0**. This update introduces new features, enhancements, and community-driven improvements that make deploying mlflow on Kubernetes easier and more efficient.

## What's New in mlflow 1.8.6 chart

The latest release includes updates to the Helm chart (version 1.8.6) as well as the underlying application (version 3.14.0). Here are the highlights:

- **Changed**: Updated the burakince/mlflow image version to 3.14.0
    - [Upstream Project](https://hub.docker.com/r/burakince/mlflow)
- **Changed**: Upgraded the postgresql dependency from 18.7.0 to 18.7.5
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a detailed overview of all changes, check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/mlflow-1.8.6) on GitHub.
