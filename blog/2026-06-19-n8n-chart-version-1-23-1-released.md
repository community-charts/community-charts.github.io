# n8n chart version 1.23.1 released!

We are excited to share the launch of the **n8n Helm chart version 1.23.1** on June 19, 2026, featuring **app version 2.26.8**. This update introduces new features, enhancements, and valuable community-driven improvements to streamline the deployment of n8n on Kubernetes.

## What's New in the n8n 1.23.1 chart

The latest release includes updates to both the Helm chart (version 1.23.1) and the underlying application (version 2.26.8). Here’s a summary of what’s new:

- **Changed**: Updated the n8nio/n8n image version to 2.26.8
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Fixed**: Resolved broken screenshot URL(s) in the n8n chart

For a comprehensive list of changes, please view the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.23.1) on GitHub.
