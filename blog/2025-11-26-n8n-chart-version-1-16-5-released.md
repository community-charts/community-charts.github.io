# n8n chart version 1.16.5 released!

We’re excited to share that the latest version of the **n8n Helm chart (v1.16.5)** is now available as of 2025-11-26. This release includes **n8n application version 1.121.2**, bringing valuable updates, dependency upgrades, and community-backed enhancements to improve your deployment experience on Kubernetes.

## What’s New in n8n Chart v1.16.5

Here’s a quick look at what’s included in this release:

- 📦 **App Update**: Upgraded to n8n version 1.121.2  
  ➤ [View the upstream project](https://github.com/n8n-io/n8n)
- 🔧 **Redis Helm Chart Updated**: From version 23.2.12 to 24.0.0  
  ➤ [Bitnami Redis on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)
- 🗃️ **PostgreSQL Helm Chart Updated**: From version 18.1.10 to 18.1.13  
  ➤ [Bitnami PostgreSQL on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For the full list of changes, check out the official [release notes on GitHub](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.5).
