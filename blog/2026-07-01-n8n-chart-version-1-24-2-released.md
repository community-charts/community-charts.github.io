# n8n chart version 1.24.2 released!

We are excited to share that the **n8n Helm chart version 1.24.2** has been officially released on 2026-07-01, featuring **app version 2.28.3**. This update introduces new features, enhancements, and valuable contributions from our community, all aimed at simplifying the deployment of n8n on Kubernetes.

## What's New in n8n 1.24.2 chart

In this release, you will find key updates to both the Helm chart (version 1.24.2) and the underlying application (version 2.28.3). Here’s what’s included:

- **Changed**: Updated the n8nio/n8n image version to 2.28.3
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a complete overview of all changes, please check the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.2) on GitHub.
