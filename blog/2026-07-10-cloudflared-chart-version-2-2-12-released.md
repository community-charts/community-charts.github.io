# Cloudflared Chart Version 2.2.12 Released!

We are excited to share the release of the **cloudflared Helm chart version 2.2.12** on July 10, 2026, featuring **app version 2026.7.1**. This update introduces new features, enhancements, and community-driven improvements aimed at simplifying the deployment of cloudflared on Kubernetes.

## What's New in Cloudflared 2.2.12 Chart

This release brings important updates to both the Helm chart (version 2.2.12) and the underlying application (version 2026.7.1). Below are the key changes included in this update:

- **Changed**: Updated cloudflare/cloudflared image version to 2026.7.1
    - [Upstream Project](https://hub.docker.com/r/cloudflare/cloudflared)

For a comprehensive list of changes, feel free to check the [release notes](https://github.com/community-charts/helm-charts/releases/tag/cloudflared-2.2.12) on GitHub.
