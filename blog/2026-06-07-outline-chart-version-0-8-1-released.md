# Outline Chart Version 0.8.1 Released!

We are excited to share the release of the **Outline Helm Chart Version 0.8.1** on June 7, 2026, which comes with the latest **App Version 1.8.1**. This update introduces an array of new features, enhancements, and community-inspired improvements that make deploying Outline on Kubernetes easier than ever.

## What's New in Outline 0.8.1 Chart

This release includes significant updates to both the Helm chart (Version 0.8.1) and the associated application (Version 1.8.1). Here’s a detailed look at the changes:

- **Updated**: The Outline image version has been upgraded to 1.8.1
    - [Upstream Project](https://hub.docker.com/r/outlinewiki/outline)
- **Updated**: Redis dependency has been refreshed from 25.5.3 to 27.0.4
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)
- **Updated**: PostgreSQL dependency has been updated from 18.7.0 to 18.7.2
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a complete overview of the updates, check out the full [release notes](https://github.com/community-charts/helm-charts/releases/tag/outline-0.8.1) on GitHub.
