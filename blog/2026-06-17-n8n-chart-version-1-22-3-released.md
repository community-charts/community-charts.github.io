# n8n chart version 1.22.3 released!

We are excited to share that the **n8n Helm chart version 1.22.3** has just been released on June 17, 2026, complete with **app version 2.26.4**! This latest update brings a host of new features, enhancements, and community-driven improvements designed to simplify your experience deploying n8n on Kubernetes.

## What's New in n8n 1.22.3 chart

With this release, we have updated both the Helm chart (version 1.22.3) and the underlying application (version 2.26.4). Here’s a summary of the key updates:

- **Changed**: The n8nio/n8n image version has been updated to 2.26.4
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: The redis dependency has been upgraded from version 27.0.4 to 27.0.10
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)
- **Changed**: The postgresql dependency has been upgraded from version 18.7.3 to 18.7.5
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a detailed list of changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.22.3) on GitHub.
