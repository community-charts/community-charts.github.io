# n8n chart version 1.24.8 released!

We are excited to share the release of the **n8n Helm chart version 1.24.8** on July 9, 2026, which features **app version 2.29.8**. This update introduces new features, enhancements, and community-driven improvements to make deploying n8n on Kubernetes easier than ever!

## What's New in n8n 1.24.8 chart

This release brings significant updates to the Helm chart (version 1.24.8) along with the underlying application (version 2.29.8). Here’s what you can expect:

- **Changed**: Updated n8nio/n8n image version to 2.29.8
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a complete list of changes, check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.8) on GitHub.
