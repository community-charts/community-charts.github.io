# n8n chart version 1.22.2 released!

We are excited to announce the release of the **n8n Helm chart version 1.22.2** on June 11, 2026, which includes **app version 2.25.7**. This latest update introduces new features, enhancements, and community-driven improvements to make deploying n8n on Kubernetes even more seamless.

## What's New in n8n 1.22.2 chart

This release encompasses updates to both the Helm chart (version 1.22.2) and the underlying application (version 2.25.7). Here’s a brief overview of the changes:

- **Changed**: Updated the n8nio/n8n image to version 2.25.7
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a comprehensive list of changes, please refer to the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.22.2) on GitHub.
