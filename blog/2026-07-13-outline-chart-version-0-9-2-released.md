# Outline Chart Version 0.9.2 Released!

We are excited to share that the **Outline Helm chart version 0.9.2** has officially launched on July 13, 2026, featuring **app version 1.9.1**. This update introduces a variety of new features, enhancements, and valuable improvements driven by our vibrant community, all aimed at streamlining the deployment of Outline on Kubernetes.

## What's New in Outline 0.9.2 Chart

This release highlights updates to both the Helm chart (version 0.9.2) and the underlying application (version 1.9.1). Here’s a snapshot of the key changes:

- **Changed**: Updated the outlinewiki/outline image version to 1.9.1
    - [Explore Upstream Project](https://hub.docker.com/r/outlinewiki/outline)
- **Changed**: Updated dependency redis from 27.0.14 to 27.0.15
    - [Check on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)

For a comprehensive list of all updates, please visit the [release notes](https://github.com/community-charts/helm-charts/releases/tag/outline-0.9.2) on GitHub.
