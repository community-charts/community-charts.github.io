# n8n chart version 1.16.13 released!

We’re excited to announce the launch of **n8n Helm chart version 1.16.13**, released on December 23, 2025. This chart now supports **app version 2.1.2**, bringing new capabilities, performance updates, and seamless Kubernetes deployment improvements.

## What’s New in Helm Chart 1.16.13

This release introduces important component updates and dependency refinements for a more stable and powerful deployment experience. Here's what's included:

- ✅ **Upgraded**: n8n image to version 2.1.2  
  ➤ [Upstream GitHub Repository](https://github.com/n8n-io/n8n)

- 🔄 **Updated**: Redis dependency from 24.0.8 to 24.1.0  
  ➤ [View Redis on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)

- 🔄 **Updated**: PostgreSQL dependency from 18.1.14 to 18.1.15  
  ➤ [View PostgreSQL on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For the full list of enhancements and changes, check out the [release notes on GitHub](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.13).
