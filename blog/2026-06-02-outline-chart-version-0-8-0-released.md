# Outline Chart Version 0.8.0 Released!

We are excited to share the release of the **Outline Helm Chart Version 0.8.0** on June 2, 2026, featuring **App Version 1.8.0**. This latest update introduces new functionalities, improvements, and valuable enhancements driven by our community, making it even easier to deploy Outline on Kubernetes.

## What's New in the Outline 0.8.0 Chart

The release includes important updates to both the Helm Chart (version 0.8.0) and the underlying application (version 1.8.0). Here’s a brief overview of the changes:

- **Added**: Support for setting custom labels on the local persistence PVC.

For a comprehensive list of updates, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/outline-0.8.0) on GitHub.
