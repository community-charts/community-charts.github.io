# n8n Chart Version 1.16.46 Released!

We are excited to announce the launch of the **n8n Helm Chart Version 1.16.46** on June 2, 2026, featuring **app version 2.23.2**. This release introduces new features, enhancements, and community-driven improvements, designed to simplify your n8n deployment on Kubernetes.

## What's New in the n8n 1.16.46 Chart

This release includes updates to both the Helm chart (version 1.16.46) and the underlying application (version 2.23.2). Here’s a look at the key changes:

- **Updated**: n8nio/n8n image version updated to 2.23.2
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Updated**: Redis dependency upgraded from version 25.5.3 to 27.0.0
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)

For a comprehensive overview of all changes, please check the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.46) on GitHub.
