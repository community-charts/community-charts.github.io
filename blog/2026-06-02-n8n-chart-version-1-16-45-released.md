# n8n chart version 1.16.45 released!

We are excited to share that the **n8n Helm chart version 1.16.45** has been officially released as of June 2, 2026, featuring **app version 2.22.6**. This latest update introduces a host of new features, enhancements, and community-driven improvements designed to streamline the deployment of n8n on Kubernetes.

## What's New in n8n 1.16.45 chart

The release of the n8n chart version 1.16.45 includes notable updates to the Helm chart alongside the underlying application version 2.22.6. Here’s a brief overview of the key changes:

- **Changed**: Updated the n8nio/n8n image version to 2.22.6
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Upgraded the PostgreSQL dependency from version 18.6.8 to 18.7.0
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For more detailed information, please check out the complete [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.45) on GitHub.
