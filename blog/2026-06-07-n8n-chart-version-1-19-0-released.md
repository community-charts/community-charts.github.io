# n8n chart version 1.19.0 released!

We are excited to share the launch of the **n8n Helm chart version 1.19.0** on 2026-06-07, featuring **app version 2.23.4**. This update introduces new features, enhancements, and valuable community-driven improvements to streamline your deployment of n8n on Kubernetes.

## What's New in n8n 1.19.0 chart

With this release, we have made notable updates to the Helm chart (version 1.19.0) and the application itself (version 2.23.4). Here’s a highlight of what’s included:

- **Added**: Configurable secret key names for external PostgreSQL and Redis existing secrets.
  
For a comprehensive list of changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.19.0) on GitHub.
