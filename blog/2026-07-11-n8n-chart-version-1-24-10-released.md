# n8n chart version 1.24.10 released!

We're excited to share the release of the **n8n Helm chart version 1.24.10** on July 11, 2026, which includes **app version 2.29.10**. This latest update introduces new features, improvements, and community-driven enhancements, making it easier than ever to deploy n8n on Kubernetes.

## What's New in n8n 1.24.10 chart

This release involves updates to both the Helm chart (version 1.24.10) and the underlying application (version 2.29.10). Here are the details:

- **Changed**: Updated the n8nio/n8n image version to 2.29.10
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Upgraded dependency redis from 27.0.14 to 27.0.15
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)

For a full list of changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.10) on GitHub.
