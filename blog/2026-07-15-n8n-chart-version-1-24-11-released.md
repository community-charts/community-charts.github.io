# n8n chart version 1.24.11 released!

We are excited to share the release of the **n8n Helm chart version 1.24.11** on July 15, 2026, featuring **app version 2.30.4**. This latest update introduces a host of new features, enhancements, and community-driven improvements designed to streamline the deployment of n8n on Kubernetes.

## What's New in n8n 1.24.11 chart

This update includes significant revisions to both the Helm chart (version 1.24.11) and the underlying application (version 2.30.4). Here’s a brief overview of the key changes:

- **Changed**: Updated the n8nio/n8n image version to 2.30.4
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a comprehensive list of changes, you can view the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.11) on GitHub.
