# n8n chart version 1.23.4 released!

We are excited to share the launch of the **n8n Helm chart version 1.23.4** on June 25, 2026, highlighting **app version 2.27.4**. This update introduces a host of innovative features and enhancements, all crafted with contributions from our dedicated community, to streamline the deployment of n8n on Kubernetes.

## What's New in n8n 1.23.4 chart

This release encompasses updates not only to the Helm chart (version 1.23.4) but also to the application itself (version 2.27.4). Check out the key changes below:

- **Changed**: Updated the n8nio/n8n image version to 2.27.4
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Fixed**: Resolved broken screenshot URL(s) in the n8n chart

For a complete overview of all changes, please refer to the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.23.4) on GitHub.
