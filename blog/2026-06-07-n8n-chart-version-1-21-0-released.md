# n8n chart version 1.21.0 released!

We are excited to announce that **n8n Helm chart version 1.21.0** has been released as of 2026-06-07, featuring **app version 2.23.4**. This update introduces new features, enhancements, and community-driven improvements designed to streamline your deployment of n8n on Kubernetes.

## What's New in n8n 1.21.0 chart

This release brings important updates to both the Helm chart (version 1.21.0) and the underlying application (version 2.23.4). Here are some highlights from the changelog:

- **Added**: Support for extra manifests through `extraManifests` and `extraTemplateManifests` values.

For a comprehensive list of changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.21.0) on GitHub.
