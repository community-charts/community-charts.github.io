# n8n chart version 1.17.1 released!

We are excited to share the news about the release of the **n8n Helm chart version 1.17.1** on June 6, 2026, featuring **app version 2.23.4**. This update introduces a range of new features, enhancements, and community-driven improvements that streamline the deployment of n8n on Kubernetes.

## What's New in n8n 1.17.1 Chart

This release includes important updates to the Helm chart (version 1.17.1) and the underlying application (version 2.23.4). Here are the key changes:

- **Changed**: Updated `n8nio/n8n` image version to 2.23.4
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Updated dependency on Redis from 27.0.0 to 27.0.4
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)
- **Changed**: Updated dependency on PostgreSQL from 18.7.0 to 18.7.2
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a comprehensive list of changes, please refer to the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.17.1) on GitHub.
