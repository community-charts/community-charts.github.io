# n8n chart version 1.16.48 released!

We are excited to announce the release of the **n8n Helm chart version 1.16.48** on June 5, 2026, featuring **app version 2.23.3**. This update introduces a range of new features, enhancements, and community-driven improvements to make your deployment of n8n on Kubernetes even smoother.

## What's New in n8n 1.16.48 chart

This release encompasses updates to the Helm chart (version 1.16.48) along with the underlying application (version 2.23.3). Here’s a glimpse of the changes included:

- **Fixed**: The unhealthyPodEvictionPolicy is now configurable on PodDisruptionBudgets for main, worker, and webhook nodes (defaults to AlwaysAllow, requires Kubernetes >= 1.26).

For a detailed list of changes, please view the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.48) on GitHub.
