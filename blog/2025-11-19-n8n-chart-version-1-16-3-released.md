# n8n chart version 1.16.3 released

We’re excited to share that the latest release of the n8n Helm chart, version 1.16.3, is now available as of November 19, 2025. This update includes the latest **n8n app version 1.120.3**, delivering enhanced performance and features thanks to our supportive community.

## Highlights of the 1.16.3 Release

This release refreshes both the Helm chart and the application version to bring you a better deployment experience on Kubernetes. Key changes include:

- 📦 Updated n8n image to version **1.120.3**  
  → [View on GitHub](https://github.com/n8n-io/n8n)
- 🛠️ Bumped PostgreSQL dependency from **18.1.9** to **18.1.10**  
  → [ArtifactHub listing](https://artifacthub.io/packages/helm/bitnami/postgresql)

Explore the full details in our [official release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.3).
