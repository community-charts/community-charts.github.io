# n8n chart version 1.24.5 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.5** on July 4, 2026, which includes **app version 2.28.6**. This release introduces a range of new features, enhancements, and community-driven improvements, making it easier than ever to deploy n8n on Kubernetes.

## What's New in the n8n 1.24.5 chart

This update brings you the latest changes in the Helm chart (version 1.24.5) alongside the underlying application (version 2.28.6). Here’s what you can expect:

- **Changed**: Updated the n8nio/n8n image version to 2.28.6
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a complete overview of all changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.5) on GitHub.
