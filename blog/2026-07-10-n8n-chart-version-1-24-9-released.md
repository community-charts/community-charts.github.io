# n8n chart version 1.24.9 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.9** on July 10, 2026, featuring the **app version 2.29.9**. This release introduces a variety of new features and enhancements, all crafted with input from our vibrant community to streamline your n8n deployment on Kubernetes.

## What's New in n8n 1.24.9 chart

The latest release encompasses vital updates to both the Helm chart (version 1.24.9) and the underlying application (version 2.29.9). Here are the highlights from the changelog:

- **Changed**: Updated the n8nio/n8n image to version 2.29.9
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Upgraded the PostgreSQL dependency from version 18.7.12 to 18.7.13
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a comprehensive overview of all the changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.9) on GitHub.
