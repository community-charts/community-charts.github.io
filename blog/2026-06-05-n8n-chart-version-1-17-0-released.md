# n8n chart version 1.17.0 released!

We are excited to share the release of the **n8n Helm chart version 1.17.0** on June 5, 2026, featuring **app version 2.23.3**. This update introduces new features, enhancements, and community-driven improvements, making it even easier to deploy n8n on Kubernetes.

## What's New in n8n 1.17.0 chart

This release includes updates to both the Helm chart (version 1.17.0) and the underlying application (version 2.23.3). Here are the highlights from the changelog:

- **Added**: Introduced a new top-level `revisionHistoryLimit` value that controls the number of old ReplicaSets/revisions retained for rollback.

For a comprehensive list of changes, check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.17.0) on GitHub.
