# Cloudflared Chart Version 2.2.14 Released!

We are excited to announce the **release of the cloudflared Helm chart version 2.2.14** on 2026-07-24, which includes **app version 2026.7.3**. This update introduces a host of new features, improvements, and community-driven enhancements that streamline the deployment of cloudflared on Kubernetes.

## What's New in Cloudflared 2.2.14 Chart

This recent release encompasses updates to both the Helm chart (version 2.2.14) and the underlying application (version 2026.7.3). Here’s a quick look at the key changes:

- **Changed**: Updated the cloudflare/cloudflared image version to 2026.7.3
    - [Upstream Project](https://hub.docker.com/r/cloudflare/cloudflared)

For a comprehensive list of changes, feel free to check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/cloudflared-2.2.14) on GitHub.
