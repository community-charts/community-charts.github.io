# n8n chart version 1.22.6 released!

We are excited to share the arrival of **n8n Helm chart version 1.22.6** on 2026-06-19, which includes the latest **app version 2.26.7**. This update features a variety of enhancements and community-driven improvements aimed at simplifying the deployment of n8n on Kubernetes.

## What's New in n8n 1.22.6 Chart

This release encompasses updates for both the Helm chart (version 1.22.6) and the underlying application (version 2.26.7). Here’s a glimpse into the changes made:

- **Changed**: Updated the n8nio/n8n image version to 2.26.7
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Updated dependency postgresql from 18.7.5 to 18.7.6
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a comprehensive list of the changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.22.6) on GitHub.
