# Cloudflared Chart Version 2.2.8 Released!

We are excited to share the release of the **cloudflared Helm chart version 2.2.8** on May 29, 2026, featuring **app version 2026.5.2**. This update introduces new features and community-driven enhancements to streamline the deployment of cloudflared on Kubernetes.

## What's New in Cloudflared 2.2.8 Chart

This release not only updates the Helm chart (version 2.2.8) but also the underlying application (version 2026.5.2). Here are the key changes:

- **Changed**: Updated cloudflare/cloudflared image version to 2026.5.2
    - [Upstream Project](https://hub.docker.com/r/cloudflare/cloudflared)

For a complete breakdown of changes, please refer to the [release notes](https://github.com/community-charts/helm-charts/releases/tag/cloudflared-2.2.8) on GitHub.
