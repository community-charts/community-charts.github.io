# n8n chart version 1.24.12 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.12** as of July 16, 2026, featuring **app version 2.30.5**. This update is packed with new features, enhancements, and community-driven improvements, all designed to streamline your deployment of n8n on Kubernetes.

## What's New in n8n 1.24.12 chart

This release includes updates to the Helm chart (version 1.24.12) along with the underlying application (version 2.30.5). Here’s a brief overview of the changes:

- **Changed**: Updated n8nio/n8n image version to 2.30.5
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Upgraded dependency postgresql from 18.7.13 to 18.8.0
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a comprehensive list of all changes, please view the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.12) on GitHub.
