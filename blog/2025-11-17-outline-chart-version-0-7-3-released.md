# Outline Chart Version 0.7.3 Released

We’re excited to announce the release of the latest update to the Outline Helm chart — version 0.7.3 — available as of November 17, 2025. This release brings support for Outline application version 1.1.0 and includes several improvements aimed at streamlining your Kubernetes deployment experience.

## What’s New in Chart Version 0.7.3

This release introduces meaningful enhancements both to the Helm chart and to the application itself. Here’s what’s been updated:

- 🛠️ Updated the container image to outlinewiki/outline app version 1.1.0  
  → [View on Docker Hub](https://hub.docker.com/r/outlinewiki/outline)
- 🔄 Refreshed Redis dependency from version 23.2.2 to 23.2.12  
  → [See on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)
- 🔄 Upgraded PostgreSQL dependency from version 18.1.3 to 18.1.9  
  → [See on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a full breakdown of changes, head over to the official [release notes](https://github.com/community-charts/helm-charts/releases/tag/outline-0.7.3) on GitHub.
