# n8n chart version 1.16.10 released!

We're excited to announce the release of the **n8n Helm chart version 1.16.10**, published on 2025-12-11! This update includes **n8n app version 1.123.5** and delivers enhanced features, updated dependencies, and community-powered improvements—making your Kubernetes deployments even smoother.

## What’s New in 1.16.10

Here's an overview of the key updates in this release:

- 🔄 **Updated Image**: Bumped the `n8nio/n8n` container image to version **1.123.5**  
  → [View upstream project](https://github.com/n8n-io/n8n)
- 🛠️ **Dependency Update**: Upgraded Redis from `24.0.0` to `24.0.4`  
  → [View Redis chart on ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)

For the full changelog, check out the official release notes on [GitHub](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.10).
