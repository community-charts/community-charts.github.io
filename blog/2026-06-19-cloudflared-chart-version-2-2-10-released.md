# Cloudflared Chart Version 2.2.10 Released!

We are excited to announce the launch of the **cloudflared Helm chart version 2.2.10**, released on 2026-06-19, featuring **app version 2026.6.1**. This update introduces a host of new features, enhancements, and community-driven improvements designed to streamline the deployment of cloudflared on Kubernetes.

## What's New in Cloudflared 2.2.10 Chart

In this release, we've made updates to both the Helm chart (version 2.2.10) and the underlying application (version 2026.6.1). Here’s a brief overview of the notable changes:

- **Changed**: Updated the cloudflare/cloudflared image version to 2026.6.1.
    - [Upstream Project](https://hub.docker.com/r/cloudflare/cloudflared)

For a detailed list of all changes, please check the [release notes](https://github.com/community-charts/helm-charts/releases/tag/cloudflared-2.2.10) on GitHub.
