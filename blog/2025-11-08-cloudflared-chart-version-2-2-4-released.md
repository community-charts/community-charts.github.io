# cloudflared chart version 2.2.4 released

We’re excited to announce the release of the **cloudflared Helm chart version 2.2.4**, published on 2025-11-08. This version includes the latest **app version 2025.11.1**, featuring valuable enhancements and updates inspired by contributions from our open-source community.

## What’s New in cloudflared 2.2.4

This update brings improvements to both the Helm chart (version 2.2.4) and the underlying application (version 2025.11.1). Highlights include:

- ✅ **Updated** cloudflare/cloudflared container image to version 2025.11.1  
  🔗 [View on Docker Hub](https://hub.docker.com/r/cloudflare/cloudflared)

For full details, check out the official [release notes](https://github.com/community-charts/helm-charts/releases/tag/cloudflared-2.2.4) on GitHub.
