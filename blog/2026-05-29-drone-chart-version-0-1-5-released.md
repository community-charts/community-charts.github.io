# Exciting Release of Drone Chart Version 0.1.5

We are excited to share that the **drone Helm chart version 0.1.5** is now live as of 2026-05-29, featuring **app version 2.28.2**. This update brings a host of new features, enhancements, and community-driven improvements that make deploying Drone on Kubernetes even more straightforward and efficient.

## What's New in Drone 0.1.5 Chart

This latest release includes updates to both the Helm chart (version 0.1.5) and the underlying application (version 2.28.2). Here’s a quick overview of what’s new:

- **Changed**: Updated to the drone/drone image version 2.28.2 
    - [Upstream Project](https://hub.docker.com/r/drone/drone)

To see the full list of changes, feel free to check the [release notes](https://github.com/community-charts/helm-charts/releases/tag/drone-0.1.5) on GitHub.
