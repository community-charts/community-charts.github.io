# n8n chart version 1.16.49 released!

We are excited to announce the release of the **n8n Helm chart version 1.16.49** on June 5, 2026, featuring **app version 2.23.3**. This update introduces a variety of new features, enhancements, and improvements driven by our vibrant community, designed to make deploying n8n on Kubernetes even easier.

## What’s New in the n8n 1.16.49 Chart

This version includes updates to both the Helm chart (1.16.49) and the underlying application (2.23.3). Here’s a glimpse of what’s included in this release:

- **Added**: Missing webhook URL to worker
    - [GitHub Issue](https://github.com/community-charts/helm-charts/issues/406)

For a comprehensive look at all the changes, check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.16.49) on GitHub.
