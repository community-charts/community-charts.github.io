# n8n chart version 1.24.15 released!

We are excited to share that the **n8n Helm chart version 1.24.15** has been officially released on 2026-07-21, featuring **app version 2.30.8**. This update introduces an array of new features, enhancements, and valuable community-driven improvements, making it easier than ever to deploy n8n on Kubernetes.

## What's New in n8n 1.24.15 chart

The latest release includes crucial updates to both the Helm chart (version 1.24.15) and the underlying application (version 2.30.8). Here’s a quick look at what’s new:

- **Changed**: Updated the n8nio/n8n image version to 2.30.8
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a complete list of changes, don’t miss our [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.15) on GitHub.
