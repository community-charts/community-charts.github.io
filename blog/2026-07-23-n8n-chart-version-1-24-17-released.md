# n8n chart version 1.24.17 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.17** on July 23, 2026, featuring **app version 2.31.5**. This release brings exciting new features, enhancements, and invaluable community-driven improvements, all aimed at making your deployment of n8n on Kubernetes even simpler.

## What's New in n8n 1.24.17 chart

The latest release includes updates to both the Helm chart (version 1.24.17) and the underlying application (version 2.31.5). Here’s a quick look at the key changes:

- **Changed**: Updated n8nio/n8n image version to 2.31.5
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a complete overview of all changes, please check the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.17) on GitHub.
