# mlflow chart version 1.11.0 released!

We are excited to share the release of the **mlflow Helm chart version 1.11.0** on 2026-06-21, featuring **app version 3.14.0**. This update introduces innovative features, valuable enhancements, and contributions from our dedicated community, aimed at streamlining the deployment of mlflow on Kubernetes.

## What's New in mlflow 1.11.0 chart

The latest release features updates to both the Helm chart (version 1.11.0) and the underlying application (version 3.14.0). Here is a summary of what’s new:

- **Security**: Introduced a securityContext to the ini-file-initializer and dbchecker init containers
    - [GitHub PR](https://github.com/community-charts/helm-charts/pull/433)

For an in-depth look at all changes, feel free to check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/mlflow-1.11.0) on GitHub.
