# n8n chart version 1.22.7 released!

We are excited to share the launch of the **n8n Helm chart version 1.22.7** as of June 19, 2026, which includes **app version 2.26.7**. This update introduces new features, enhancements, and valuable improvements driven by our community, making it even easier to deploy n8n on Kubernetes.

## What's New in n8n 1.22.7 chart

This release features updates to the Helm chart (version 1.22.7) and the underlying application (version 2.26.7). Here’s a glimpse of the changes:

- **Fixed**: Enhanced community node detection to support scoped community nodes (e.g., @myorg/n8n-nodes-mynode)
    - [GitHub Issue](https://github.com/community-charts/helm-charts/issues/241)

For a comprehensive list of changes, please view the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.22.7) on GitHub.
