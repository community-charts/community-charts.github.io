# n8n chart version 1.24.7 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.7** on July 8, 2026, featuring **app version 2.29.7**. This update introduces enriching features, enhancements, and community-driven improvements that make deploying n8n on Kubernetes even easier and more efficient.

## What's New in n8n 1.24.7 chart

This release encompasses updates to both the Helm chart (version 1.24.7) and the underlying application (version 2.29.7). Here are the highlights:

- **Changed**: Updated the n8nio/n8n image version to 2.29.7
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Updated dependency redis from 27.0.13 to 27.0.14
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)
- **Changed**: Updated dependency postgresql from 18.7.11 to 18.7.12
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a complete list of changes, you can view the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.7) on GitHub.
