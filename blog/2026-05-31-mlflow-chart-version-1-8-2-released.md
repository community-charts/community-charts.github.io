# mlflow chart version 1.8.2 released!

We are excited to announce the release of the **mlflow Helm chart version 1.8.2** on May 31, 2026, featuring **app version 3.12.0**! This update introduces a range of new features, enhancements, and community-driven improvements designed to simplify deploying mlflow on Kubernetes.

## What's New in mlflow 1.8.2 chart

This release includes significant updates to the Helm chart (version 1.8.2) along with the underlying application (version 3.12.0). Here’s a summary of the key changes:

- **Changed**: Updated the burakince/mlflow image version to 3.12.0
    - [Upstream Project](https://hub.docker.com/r/burakince/mlflow)
- **Changed**: Updated the dependency on PostgreSQL from version 18.1.13 to 18.6.10
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a complete list of changes, please check the [release notes](https://github.com/community-charts/helm-charts/releases/tag/mlflow-1.8.2) on GitHub.
