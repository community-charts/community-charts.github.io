# n8n chart version 1.23.3 released!

We are excited to share the release of the **n8n Helm chart version 1.23.3** on June 24, 2026, which includes the latest **app version 2.27.3**. This new release introduces a range of features, enhancements, and community-driven improvements, all designed to streamline deploying n8n on Kubernetes.

## What's New in the n8n 1.23.3 Chart

The n8n 1.23.3 chart includes important updates that enhance the overall functionality and performance. Here’s a quick overview of the changes:

- **Changed**: Updated the n8nio/n8n image to version 2.27.3.
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Upgraded the PostgreSQL dependency from version 18.7.7 to 18.7.8.
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a comprehensive view of all changes, please check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.23.3) on GitHub.
