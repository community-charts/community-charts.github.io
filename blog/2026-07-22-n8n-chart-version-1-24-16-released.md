# n8n chart version 1.24.16 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.16** on July 22, 2026, featuring **app version 2.31.4**. This update introduces a range of new features, enhancements, and community-driven improvements aimed at making the deployment of n8n on Kubernetes more user-friendly and efficient.

## What's New in n8n 1.24.16 chart

This release encompasses updates to both the Helm chart (version 1.24.16) and the underlying application (version 2.31.4). Here are the key changes:

- **Changed**: Updated the n8nio/n8n image version to 2.31.4.
    - [Upstream Project](https://github.com/n8n-io/n8n)

For a comprehensive overview of the changes, please check the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.16) on GitHub.
