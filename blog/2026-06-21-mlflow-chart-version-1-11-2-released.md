# mlflow chart version 1.11.2 released!

We’re excited to share the release of the **mlflow Helm chart version 1.11.2** on June 21, 2026, featuring **app version 3.14.0**. This update introduces new features, enhancements, and community-driven improvements to streamline the deployment of mlflow on Kubernetes.

## What's New in mlflow 1.11.2 chart

This latest version includes important updates to the Helm chart (version 1.11.2) alongside the underlying application (version 3.14.0). Here’s a brief overview of the enhancements:

- **Changed**: Updated dbchecker image version to 1.38.0
    - [dbchecker](https://hub.docker.com/r/library/busybox)
- **Changed**: Updated ini-file-initializer image version to 1.38.0
    - [ini-file-initializer](https://hub.docker.com/r/library/busybox)

For a comprehensive list of changes, feel free to check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/mlflow-1.11.2) on GitHub.
