# n8n chart version 1.24.6 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.6** on July 7, 2026, featuring **app version 2.28.7**. This release is packed with new features, enhancements, and community-driven improvements aimed at making your n8n deployment on Kubernetes even easier!

## What's New in n8n 1.24.6 chart

This update introduces significant enhancements to both the Helm chart (version 1.24.6) and the underlying application (version 2.28.7). Here’s a glimpse of what’s included:

- **Changed**: Updated n8nio/n8n image version to 2.28.7
    - [Upstream Project](https://github.com/n8n-io/n8n)

To see all the changes in detail, check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.6) on GitHub.
