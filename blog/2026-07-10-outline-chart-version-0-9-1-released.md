# Outline Chart Version 0.9.1 Released!

We are excited to share that the **Outline Helm Chart version 0.9.1** has officially launched on July 10, 2026, featuring **app version 1.9.0**. This update introduces a host of new features, enhancements, and community-driven improvements designed to make deploying Outline on Kubernetes smoother and more efficient.

## What's New in Outline 0.9.1 Chart

This release includes crucial updates for both the Helm chart (version 0.9.1) and the underlying application (version 1.9.0). Here are the highlights from the changelog:

- **Updated**: The `outlinewiki/outline` image version has been upgraded to **1.9.0**
    - [Upstream Project](https://hub.docker.com/r/outlinewiki/outline)
- **Updated**: Redis dependency has been bumped from **27.0.4** to **27.0.14**
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/redis)
- **Updated**: PostgreSQL dependency has been refined from **18.7.2** to **18.7.13**
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For an extensive overview of changes, please refer to the [release notes](https://github.com/community-charts/helm-charts/releases/tag/outline-0.9.1) on GitHub.
