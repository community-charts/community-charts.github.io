# n8n chart version 1.24.3 released!

We are excited to share the launch of the **n8n Helm chart version 1.24.3** on 2026-07-02, which includes **app version 2.28.4**. This update introduces a variety of new features, enhancements, and community-driven improvements aimed at making your n8n deployment on Kubernetes even smoother.

## What's New in n8n 1.24.3 chart

This release encompasses updates to both the Helm chart (version 1.24.3) and the underlying application (version 2.28.4). Here’s a quick overview of the changes:

- **Changed**: Updated n8nio/n8n image version to 2.28.4
    - [Upstream Project](https://github.com/n8n-io/n8n)
- **Changed**: Updated dependency postgresql from 18.7.9 to 18.7.10
    - [ArtifactHub](https://artifacthub.io/packages/helm/bitnami/postgresql)

For a comprehensive list of all changes, check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/n8n-1.24.3) on GitHub.
