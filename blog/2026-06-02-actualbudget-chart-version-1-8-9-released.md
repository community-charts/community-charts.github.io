# actualbudget chart version 1.8.9 released

We are excited to share the release of the **actualbudget Helm chart version 1.8.9** as of June 2, 2026, along with **app version 26.6.0**. This latest update introduces new features, enhancements, and improvements driven by our wonderful community, all designed to make deploying actualbudget on Kubernetes smoother and more efficient.

## What's New in actualbudget 1.8.9 chart

The 1.8.9 release includes significant updates to both the Helm chart and the application. Here’s a quick overview of what has changed:

- **Changed**: Updated the actualbudget/actual-server image version to 26.6.0

For a comprehensive list of changes, you can check out the [release notes](https://github.com/community-charts/helm-charts/releases/tag/actualbudget-1.8.9) on GitHub.
